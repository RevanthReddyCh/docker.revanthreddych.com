Docker Installation on Ubuntu

Welcome to the Docker installation guide for Ubuntu.

Step-by-Step Installation Guide

  • Install Docker CE (Community Edition)
  • Install Docker Compose

Install Docker CE

Follow these steps to install Docker Community Edition (CE) on Ubuntu:

  1. Update your package index:

    sudo apt-get update

  2. Install the required packages to allow apt to use a repository over HTTPS:

    sudo apt-get install apt-transport-https ca-certificates curl software-properties-common
  3. Add Docker's official GPG key:
    4. curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -
  4. Add the Docker repository to APT sources:
    5. sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable"
  5. Update the package index again:
    6. sudo apt-get update
  6. Start the Docker service:
    7. systemctl start docker
  7. Verify the Docker service is up and running: 
    8. systemctl status docker
  8. Install Docker CE:
    9. sudo apt-get install docker-ce
  9. Verify that Docker CE is installed correctly by running the hello-world container:
    10. sudo docker run hello-world

Install Docker Compose

To install Docker Compose, follow these steps:

  1. Download the Docker Compose binary into the `/usr/local/bin` directory:
    2. sudo curl -L "https://github.com/docker/compose/releases/download//docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
  2. Apply executable permissions to the binary:
    3. sudo chmod +x /usr/local/bin/docker-compose
  3. Verify the installation by checking the version:
    4. docker-compose --version
